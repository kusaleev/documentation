Струкрутра репозитория имеет следующий вид ()

[source,markdown]
----
  src
    └── docs
        ├── asciidoc
        │   ├── examples
        │   │   ├── REST
        │   │   │   └── method-1
        │   │   │       ├── method-1-request.adoc
        │   │   │       └── method-1-response.adoc
        │   │   └── SOAP
        │   │       └── external-SOAP-service
        │   │           ├── external-SOAP-service-request.adoc
        │   │           └── external-SOAP-service-response.adoc
        │   ├── method-1.adoc
        │   └── method-2.adoc
        └── index.adoc
----
